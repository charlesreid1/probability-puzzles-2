# Probability Puzzles 2

A collection of programming solutions and simulations for probability puzzles drawn from Donald Knuth's *The Art of Computer Programming* (TAOCP). Each puzzle wraps a classic algorithmic concept in a narrative story, and each script verifies the analytical solution through Monte Carlo simulation.

## **Course I: The Theorems**

### **Permutations & Sorting**
* [The Careless Typesetter's Toll](scripts/careless_typesetter.py) (Problem 1) - Expected number of inversions (adjacent swaps) in a random permutation of N pages.

### **Stream Selection**
* [The Cautious Alchemist's Vial](scripts/cautious_alchemist.py) (Problem 2) - Reservoir sampling: maintaining a uniform random sample from a stream of unknown length.

### **Binary Search Trees & Quicksort**
* [The Fractious Tour Guide](scripts/fractious_tour_guide.py) (Problem 3) - Expected depth of the k-th element in a random BST (comparisons in Quicksort).

### **Hash Tables & Linear Probing**
* [The Infinite Hashway](scripts/infinite_hashway.py) (Problem 4) - Expected probe length when inserting into a linear probing hash table.

### **Cycle Structure of Permutations**
* [King Permutio's Dungeon](scripts/king_permutio.py) (Problem 5) - Probability that a random permutation of N elements is a single cycle.

## **Course II: The Boundaries**

### **Permutations & Sorting**
* [The Archivist's Epiphany](scripts/archivists_epiphany.py) (Problem 6) - Expected number of left-to-right maxima in a random permutation.

### **Stream Selection**
* [The King's Council](scripts/kings_council.py) (Problem 7) - Generalized reservoir sampling for K items from an infinite stream.

### **Binary Search Trees & Quicksort**
* [The Indecisive General](scripts/indecisive_general.py) (Problem 8) - Expected number of leaf nodes in a random BST.

### **Hash Tables & Linear Probing**
* [The Sleepy Knights of the Round Table](scripts/sleepy_knights.py) (Problem 9) - Probability the last knight sits in his favorite chair (parking problem).

### **Cycle Structure of Permutations**
* [The Spies Who Watched Me](scripts/spies_who_watched_me.py) (Problem 10) - Expected number of cycles in a random permutation.
